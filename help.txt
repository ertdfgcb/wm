DIR is either:  h (left), j (down), k (up), or l (right)
Super is windows key or command key
  Toggle help:                  F1
  Logout:                       super + alt + escape
  Close focused window:         super + shift + w
  Switch desktop layout:        super + l
  Equalize window area:         super + b
  Reset window area:            super + shift + b
  Make window floating:         super + f
  Make window fullscreen:       super + s
  Switch to last desktop:       super + tab
  Switch to last window:        super + `
  Desktop cycle foreward:       super + ctrl + tab
  Desktop cycle backward:       super + shift + tab
  Window cycle foreward:        super + ctrl + `
  Window cycle backward:        super + shift + `
  Swap with last desktop:       super + alt + tab
  Swap with last window:        super + alt + `
  Move window focus:            super + DIR
  Move window:                  super + shift + DIR 
  Switch to desktop 0-9:        super + (0-9)
  Move window to desktop 0-9:   super + shift + (0-9)
  Move floating window:         super + drag
  Resize floating window:       super + ctrl + drag
  Add desktop:                  super + d
  Remove desktop:               super + alt + d
  Remove all empty desktops:    super + shift +d
  Rename desktop:               super + r
  Append line to autorun file:  super + a
  Launch terminal:              super + return
  Launch program:               super + space
  Set up window and edge gaps:  super + shift + s
  Toggle system tray:           super + t
  Toggle window title:          super + w
